Extend the String object (JS) or create a function (Python) that converts the value of 
the String to and from Base64 using the ASCII character set.

Usage:

# should return 'dGhpcyBpcyBhIHN0cmluZyEh'
to_base_64('this is a string!!')

# should return 'this is a string!!'
from_base_64('dGhpcyBpcyBhIHN0cmluZyEh')
You can learn more about Base64 encoding and decoding https://zh.wikipedia.org/wiki/Base64.
